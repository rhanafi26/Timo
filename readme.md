# Timo ⏱️

![Version](https://img.shields.io/badge/version-1.1.0-blue)
![Android](https://img.shields.io/badge/Android-3DDC84?logo=android&logoColor=white)
![Kotlin](https://img.shields.io/badge/Kotlin-7F52FF?logo=kotlin&logoColor=white)

**Timo** is an open-source Android timer application designed to help users focus better while studying and improve their learning sessions through structured time management.

> Stay focused. Upgrade your lessons. Master your time.

---

## 📌 About The Project

Timo is built for students and learners who want to:

- Improve concentration
- Structure study sessions
- Manage time effectively
- Reduce distractions
- **Connect and compete with friends** 🆕

It works as a simple and efficient study timer to help users stay productive and disciplined.

---

## 🚀 Features

### Current Features (v1.1.0)

| Feature | Description |
|---------|-------------|
| ⏲️ **Customizable timer** | Set your own study duration |
| 🎯 **Focus session tracking** | Monitor your progress over time |
| 🏆 **Leaderboard** 🆕 | View top users, read comments, add friends |
| 🔔 **Smart Notifications** 🆕 | Friend follow alerts & click-to-comment replies |
| 👥 **Social Features** 🆕 | Add friends and follow their progress |
| 📚 **Study-focused design** | Built specifically for learners |
| 📱 **Clean Android UI** | Simple, intuitive interface |
| ⚡ **Lightweight & fast** | Minimal resource usage |

### What's New in v1.1.0

✨ **Leaderboard Page**
- Browse list of top-performing players
- View and write comments
- Add friends directly from leaderboard

🔔 **Notification System**
- Pop-up notifications when clicked → Direct to leaderboard comment section
- Real-time alerts when friends follow you
